Descrição da tecnologia escolhida

Go (ou Golang) é uma linguagem de programação de código aberto criada pelo Google, conhecida por ser simples, confiável e eficiente. Ela combina a simplicidade de linguagens dinâmicas com o desempenho de linguagens compiladas, sendo ideal para o desenvolvimento de sistemas concorrentes e distribuídos. Suas principais aplicações incluem servidores web, ferramentas de automação e serviços de rede de alta performance.

Principais pontos fortes e diferenciais

  • Alto desempenho: Go é compilado diretamente para código de máquina, o que o torna extremamente rápido.
  • Simplicidade e produtividade: Sintaxe limpa e fácil de ler, facilitando o aprendizado e a manutenção do código.
  • Concorrência nativa: Através das goroutines e channels, permite lidar com múltiplas requisições simultâneas com baixo consumo de recursos.
  • Segurança e robustez: Tipagem estática e forte sistema de tratamento de erros.
  • Framework minimalista (Gin): Facilita a criação de rotas e middleware, com suporte nativo para JSON e tratamento de requisições HTTP.

Seção 2: O Código do Servidor

          
            package main
           
            import (
              "github.com/gin-gonic/gin"
            )
           
            func main() {
              route := gin.Default()
           
            
              route.GET("/", func(c *gin.Context) {
                html := ""

                c.Header("Content-Type", "text/html; charset=utf-8")
                c.String(200, html)
              })
           
              route.Run(":8080")
            }

$ go run server.go

Seção 3: Conclusão

Principal dificuldade inicial: A principal dificuldade foi compreender a estrutura de pacotes e módulos no Go e configurar corretamente o mod init e as dependências do Gin.

Recomendação final do grupo: O grupo recomenda fortemente o uso do Go + Gin para projetos que exigem rapidez, estabilidade e escalabilidade. É uma excelente opção para desenvolvedores que desejam aprender back-end de forma moderna, com sintaxe limpa e excelente performance.
