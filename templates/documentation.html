Servidor em Go (Golang) com o Framework Gin

Participantes: Bruno José Adriano, Mel Izis Godri e Pedro Padilha

Descrição da tecnologia escolhida

O Go, ou Golang, é uma linguagem de programação de código aberto desenvolvida pelo Google. Ela foi projetada para unir simplicidade e alto desempenho, oferecendo compilação rápida, tipagem estática e excelente suporte à concorrência.

O framework Gin é uma biblioteca popular do ecossistema Go para o desenvolvimento de servidores web. Ele oferece uma sintaxe limpa, tratamento de requisições HTTP, gerenciamento de rotas e suporte nativo a JSON — tudo com foco em eficiência e facilidade de uso.

Seção 1: Pontos Fortes e Filosofia

  • Performance elevada: Go é compilado em código de máquina, resultando em execução extremamente rápida e eficiente.
  • Concorrência simples: As goroutines e os channels facilitam o processamento paralelo de múltiplas requisições.
  • Sintaxe minimalista: A linguagem prioriza clareza e padronização, evitando código verboso.
  • Ecossistema moderno: O Go inclui ferramentas nativas de testes, formatação e controle de dependências.
  • Framework Gin: Oferece rotas intuitivas, middlewares leves e excelente integração com JSON.

Caso de uso ideal: APIs RESTful, microsserviços e sistemas em nuvem que exigem escalabilidade e baixa latência.

Seção 2: Pré-Requisitos e Setup

  • Instalar o Go: https://go.dev/doc/install
  • Criar o módulo do projeto: 
    $ go mod init servidor-gin
  • Instalar o framework Gin: 
    $ go get -u github.com/gin-gonic/gin
  • Criar uma pasta chamada templates e adicionar o arquivo documentation.html (esta página).
  • Executar o servidor: 
    $ go run server.go

Seção 3: Código do Servidor


package main

import (
  "github.com/gin-gonic/gin"
)

func main() {
  router := gin.Default()

  // Carrega os templates HTML da pasta "templates"
  router.LoadHTMLGlob("templates/*.html")

  // Rota principal servindo a documentação
  router.GET("/", func(c *gin.Context) {
    c.HTML(200, "documentation.html", nil)
  })

  // Executa o servidor na porta 8080
  router.Run()
}

O servidor acima utiliza o framework Gin para servir esta própria documentação na rota principal (/). Ao executar go run server.go, o sistema inicia um servidor HTTP local acessível em http://localhost:8080.

Seção 4: Conclusão

Principal dificuldade inicial: A compreensão da estrutura de pacotes e módulos do Go, junto com o carregamento de templates via Gin, exigiu prática para configurar corretamente o ambiente e as dependências.

Pontos em comuns com outras linguagens: Particularmente, sentimos muita semelhança na sintaxe com a mistura de NodeJS com EXPRESS e C++. Na hora de colocar rodar o servidor, a forma de fazer isso remete muito ao do NodeJS.

Recomendação final do grupo: O Go com o framework Gin é uma excelente escolha para quem deseja desenvolver servidores rápidos e modernos. O desempenho, a simplicidade e a clareza da linguagem tornam o processo de aprendizado produtivo e recompensador.
